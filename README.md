# `Nexmeet`

nexmeet a open source event platform

## `Installation`

To install the project dependencies, follow these steps:

1. Clone the repository:

   ```sh
   git clone https://github.com/nexmeet/nexmeet.git
   ```

2. Navigate to the project directory:

   ```sh
   cd nexmeet-platform
   ```

3. Install the dependencies:

   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

Make sure you have Node.js and npm installed on your machine.

## Usage

After starting the development server, you can access the application at `http://localhost:5173`. The main entry point of the application is defined in [main.tsx](main.tsx).

## `Scripts`

The following scripts are available in the [package.json](package.json) file:

- `dev`: Starts the development server.
- `build`: Builds the project for production.
- `lint`: Runs ESLint to check for linting errors.
- `preview`: Previews the production build.
